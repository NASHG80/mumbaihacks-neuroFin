# NeuroFin - Smart Personal Finance Platform

NeuroFin is an AI-powered financial planning platform designed to provide personalized insights for budgeting, investments, tax optimization, and financial goal tracking. With intelligent agent-based guidance, real-time analytics, and an intuitive interface, the platform helps users make confident and informed financial decisions.

 # Features

1. Agentic AI system with role-based intelligence
2. Personalized budgeting, saving, goals and investment insights
3. Tax optimization and expense classification
4. Forecasting tools and festive financial planning
5. Secure and user-friendly interface built with modern technologies

# Setup Instructions

1. Clone the repository  
```
git clone https://github.com/NASHG80/mumbaihacks-neuroFin.git
cd mumbaihacks-neuroFin
```
2. Install dependencies
```
npm install
```

3. Create Environment Variables
Create a .env file in the project root:
[refer to .env.example for .env structure]

4. Start the development server
```
npm run dev
```

5. Start backend
```
cd backend
npm install
npm run dev or npm start
```
